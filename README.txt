========================================
BRADLEY WILLIAMS — PORTFOLIO WEBSITE
========================================
 
A personal portfolio site built to showcase my work across photography,
video editing, graphic design, 3D printing, and web development. The site
is designed to serve as a living resume and creative gallery that I can
continue to expand as my body of work grows.
 
 
----------------------------------------
ABOUT THE SITE
----------------------------------------
 
This site was built from scratch using plain HTML and CSS — no frameworks,
no libraries beyond Font Awesome for icons and Google Fonts for typography.
The goal was to keep the codebase clean, lightweight, and easy to maintain.
 
Sections:
  - Hero          : Introduction with a semi-transparent background image
  - About         : Headshot, bio, and skills list
  - Web Design    : Screenshots and links to web projects
  - Editing & Photography : Video showreel and photo/design work
  - 3D Prints     : Photo gallery of printed and painted helmet replicas
  - Resume        : Education, work experience, and major projects
  - Contact       : Email, TikTok, and GitHub links
 
 
----------------------------------------
TECHNOLOGIES USED
----------------------------------------
 
  - HTML5
  - CSS3 (custom properties, grid, flexbox, animations, media queries)
  - Vanilla JavaScript (burger menu, dropdown, intersection observer)
  - Font Awesome 7  —  icons
  - Google Fonts    —  Titan One (headings), Signika (body)
 
 
----------------------------------------
AI ASSISTANCE
----------------------------------------
 
This site was built with the assistance of Claude (by Anthropic) throughout
the development process. Claude was used to:
 
  - Generate the initial HTML and CSS structure from hand-drawn sketches
  - Debug layout and formatting issues as they came up
  - Write and refine CSS for images, video, dropdowns, animations,
    and responsive design
  - Add accessibility attributes (aria labels, roles, semantic HTML)
  - Improve mobile support for the navigation menu and dropdown
  - Write the meta description and Open Graph tags
  - Generate the resume section from an uploaded PDF
  - Advise on file structure, path issues, and browser compatibility
 
While Claude assisted with code generation and debugging, all content,
creative direction, design decisions, and final edits were made by me.
The sketches the site was built from, the copy, the images, the video,
and the overall vision are entirely my own work.
